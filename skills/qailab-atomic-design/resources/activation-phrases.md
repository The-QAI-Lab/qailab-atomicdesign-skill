# Activation phrases

Use the skill when prompts contain or imply phrases like:

- Build a mobile application
- Let's build a mobile application
- Build a mobile app
- Build a website
- Let's build a website
- Design a mobile app
- Design a website
- Build a web app
- Create a dashboard UI
- Create a landing page
- Create a product interface
- Build a frontend
- Build an iOS app
- Build a SwiftUI app
- Build a React site

## Semantic triggers

Also activate when the user is clearly asking to:

- create a new app UI from scratch
- design a responsive interface
- structure a cross-platform frontend
- create reusable components for a product UI
- establish a design system before implementation

## Do not activate when

The user is only asking for:

- backend APIs only
- infrastructure only
- database schema only
- bug fixing in an existing single component with no architecture request
- analytics logic with no interface design scope
