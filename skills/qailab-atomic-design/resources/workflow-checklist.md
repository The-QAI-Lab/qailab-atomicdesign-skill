# Workflow checklist

Use this as a fast execution checklist.

## Step 1 — Scope
- Identify product type
- Identify audience
- Identify platform
- Identify screens/pages
- Identify constraints

## Step 2 — Atoms
- Tokens
- Base primitives
- States
- Accessibility notes

## Step 3 — Molecules
- Small functional groupings
- Inputs used
- Reuse opportunities

## Step 4 — Organisms
- Section-level structures
- Child composition
- Responsive behavior

## Step 5 — Templates
- Structural page/screen shells
- Region placement
- Navigation placement

## Step 6 — Pages / Screens
- User goal
- Template used
- Organisms included
- Data dependencies
- States

## Step 7 — Implementation mapping
- SwiftUI / React / HTML/CSS / hybrid mapping
- File and folder structure
- Next build step

## Gate
Do not move to code-first implementation until the hierarchy above is complete.
